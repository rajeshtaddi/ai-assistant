# AI Assistant (LangChain + Ollama + MCP)

A local AI assistant built with FastAPI, LangChain, Ollama and MCP tools.

This project demonstrates how to build an AI system with tool calling and a modern chat UI.

---

## Features

- AI Chat using local LLM
- Weather tool
- MCP server integration
- FastAPI backend
- Modern chat UI
- Tool calling using LangChain

---

## Tech Stack

- Python
- FastAPI
- LangChain
- Ollama
- MCP (Model Context Protocol)
- HTML / CSS / JavaScript
